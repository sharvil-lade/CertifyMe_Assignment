# Frontend (React + Vite)

## Run

```powershell
npm install
npm run dev
```

The dev server runs on `http://127.0.0.1:5173`.

Vite proxy forwards API requests to Flask backend on `http://127.0.0.1:5000`:

- `/signup`
- `/login`
- `/forgot-password`
- `/session`
- `/logout`
- `/opportunities*`
- `/health`
